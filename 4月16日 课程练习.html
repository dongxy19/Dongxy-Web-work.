CSS基础语法

CSS规则由选择器和声明块组成：

/* 选择器 { 属性: 值; } */
h1 {
  color: blue;
  font-size: 24px;
}

/* 多组声明 */
.container {
  width: 80%;
  margin: 0 auto;
  padding: 20px;
  background-color: #f0f0f0;
}

/* 注释方式 */
/* 这是CSS注释 */

选择器

CSS选择器用于选择需要添加样式的元素：

/* 元素选择器 */
p { color: #333; }

/* 类选择器 */
.highlight { background: yellow; }

/* ID选择器 */
#header { height: 80px; }

/* 属性选择器 */
a[target="_blank"] { color: red; }

/* 后代选择器 */
div p { margin: 10px 0; }

/* 伪类选择器 */
a:hover { text-decoration: underline; }
li:nth-child(odd) { background: #eee; }

盒模型

每个HTML元素都是一个盒子，由内容、内边距、边框和外边距组成：

.box {
  /* 内容尺寸 */
  width: 300px;
  height: 200px;

  /* 内边距 */
  padding: 20px;
  padding-top: 10px;

  /* 边框 */
  border: 2px solid #3498db;
  border-radius: 10px;

  /* 外边距 */
  margin: 30px;
  margin-bottom: 40px;
}

/* 盒模型计算方式 */
.box-sizing {
  box-sizing: border-box; /* 包括padding和border */
}

Flexbox布局

Flexbox是一种一维布局模型，用于创建灵活的响应式布局：

/* 容器属性 */
.container {
  display: flex;
  flex-direction: row; /* row | row-reverse | column | column-reverse */
  justify-content: center; /* 主轴对齐 */
  align-items: stretch; /* 交叉轴对齐 */
  flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */
  gap: 20px; /* 项目间距 */
}

/* 项目属性 */
.item {
  order: 2; /* 显示顺序 */
  flex-grow: 1; /* 放大比例 */
  flex-shrink: 1; /* 缩小比例 */
  flex-basis: 200px; /* 初始尺寸 */
  align-self: center; /* 单独对齐 */
}

Grid布局

CSS Grid是一个二维布局系统，用于创建复杂的网格布局：

/* 容器属性 */
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 三列等宽 */
  grid-template-rows: 100px auto 200px;
  gap: 20px; /* 行列间距 */
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
}

/* 项目属性 */
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

/* 项目位置 */
.item {
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 2;
  grid-row-end: 4;
}

响应式设计

响应式设计确保网站在各种设备上都能良好显示：

/* 视口设置 */
meta name="viewport" content="width=device-width, initial-scale=1"

/* 流式布局 */
.container {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto;
}

/* 响应式图片 */
img {
  max-width: 100%;
  height: auto;
}

/* 响应式单位 */
.responsive-text {
  font-size: clamp(1rem, 2.5vw, 2rem);
}

关键帧动画

使用@keyframes创建复杂的CSS动画：

/* 定义动画 */
@keyframes slideIn {
  from {
    transform: translateX(-100%);
  }
  to {
    transform: translateX(0);
  }
}

/* 应用动画 */
.element {
  animation: slideIn 1s ease-in-out forwards;
}

/* 多关键帧动画 */
@keyframes bounce {
  0% { transform: translateY(0); }
  50% { transform: translateY(-30px); }
  100% { transform: translateY(0); }
}

/* 动画属性 */
.animated {
  animation-name: bounce;
  animation-duration: 2s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}
